# 第三方组件说明

ds Eagle Tagger 的 Windows 安装包包含或会按需安装以下第三方组件：

- `uv 0.6.11`：用于创建 Python 环境和安装依赖，采用 MIT License 或 Apache License 2.0。完整许可文本位于 `eagle-plugin/engine/licenses/`。
- `SmilingWolf/wd-eva02-large-tagger-v3`：首次识别时按需下载，模型页面声明为 Apache License 2.0。
- `ONNX Runtime GPU`：MIT License。
- `NumPy`：BSD 3-Clause License。
- `Pillow`：HPND License。
- `Requests`：Apache License 2.0。

各组件的版权与商标归其各自权利人所有。本文件仅用于说明 ds Eagle Tagger 对这些组件的使用方式，不改变其原始许可条款。
