# 隐私说明

ds Eagle Tagger 是本地运行的图片标签工具。

- 图片内容、文件路径、文件夹选择和识别结果均在用户电脑上处理。
- 标签仅通过 Eagle 提供的本地接口写入当前 Eagle 图库。
- 插件没有账号系统，不包含广告、遥测、行为统计或远程日志上传。
- 插件只在安装运行组件、下载模型文件或检查模型更新时访问网络。
- 安装运行组件时可能访问中国科学技术大学镜像、清华大学镜像、GitHub 与 PyPI；来源失败后会自动切换。
- 用户也可以从网盘取得模型并在设置中本地导入；导入过程不会上传文件。
- 用户主动点击“检查模型更新”时，插件会访问配置中的批准模型清单地址；不会上传图片、标签或使用统计。
- 默认模型从 Hugging Face 下载；网络不可用时，程序可能尝试配置中的备用镜像。
- Eagle 本地接口使用 `127.0.0.1`，不会将图片发送到第三方服务器。
- 插件会在本地 SQLite 文件中记录 Eagle 图片 ID、模型版本、识别设置、上次模型生成的标签和处理时间，用于避免重复处理。
- 处理记录的保存位置可由用户在设置中改到其他硬盘，不会上传到网络。

本地文件默认保存在：

```text
%LOCALAPPDATA%\EagleAutoTagger
```

卸载插件不会自动删除该目录，以免用户重复下载大体积模型。需要彻底移除时，请先关闭插件，再手动删除该目录。
